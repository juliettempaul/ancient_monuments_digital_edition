# Vetusta Monuments: Ancient Monuments, A Digital Edition (Contributor)

This repository documents my research and data contributions to Vetusta Monumenta, a digital humanities project. My work involved preparing the entry on Plate 1.15: Tomb of Robart Colles in support of the project's broader scholarly aims.

Vetusta Monumenta is hosted externally; this repository serves as a record of my individual contributions to the project.

Go to [my contribution](https://scalar.missouri.edu/vm/vol1plate15-tomb-robart-colles).
For context, go to Noah Heringman's [introduction](https://scalar.missouri.edu/vm/introduction).
